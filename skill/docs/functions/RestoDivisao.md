# RestoDivisao

## Assinatura

```lspt
Funcao RestoDivisao (Dividendo,Divisor,Resto);
```

## Código
N/A

## Descrição

Calcula o resto da divisão (operação módulo).

## Parâmetros

- **Dividendo** - Entrada: Campo/Variável que será dividido
- **Divisor** - Entrada: Campo/Variável pelo qual o **Dividendo** será dividido
- **Resto** - Entrada: Variável que receberá o resto da divisão

## Exemplo de Uso

```lspt
Valor1 = 1500;
Valor2 = 400;
RestoDivisao (Valor1, Valor2, VResto);
@ VResto será 300. @
```
