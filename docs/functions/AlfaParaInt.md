## Descrição

Converte um valor alfanumérico para o tipo Inteiro.

## Parâmetros

- **Alfa_Origem** - Entrada: Campo/variável tipo que contém o número a ser convertido
- **Num_Destino** - Entrada: Campo/Variável tipo que receberá o resultado da conversão

## Exemplo de Uso

```lspt
Definir Alfa vaTexto;
Definir Numero vnInteiro;

vaTexto = "123";
AlfaParaInt(vaTexto, vnInteiro); @ vnInteiro será 123 @
```
