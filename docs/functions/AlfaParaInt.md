# AlfaParaInt

## Assinatura

```lspt
Funcao AlfaParaInt (Alfa_Origem,Num_Destino);
```

## Código
N/A

## Descrição

Converte um número armazenado como **Alfa** e o retorna como um tipo **Numero**. No caso de impossibilidade de conversão, o resultado será **0** (zero).

## Parâmetros

- **Alfa_Origem** - Entrada: Campo/variável tipo que contém o número a ser convertido
- **Num_Destino** - Entrada: Campo/Variável tipo que receberá o resultado da conversão

## Exemplo de Uso

```lspt
AlfaParaInt (R034Fun.CadFol,Vcadastro);
```
