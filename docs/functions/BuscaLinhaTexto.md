# BuscaLinhaTexto


## Descrição

Esta função é utilizada para imprimir as linhas do texto que a função QuebraTexto dividiu.

## Parâmetros

- **Texto** (`Alfa`) - Entrada: Será o mesmo Campo ou Variável indicado na função QuebraTexto.
- **NroLin** (`Numero`) - Entrada: Campo, Valor ou Variável que indicará qual a linha que será impressa.
- **LinTex** (`Alfa End`) - Saída: Variável que retornará a parte do Texto que será impressa.

## Valores de Retorno

- LinTex: Variável alfanumérica que retornará a parte do Texto que será impressa.
