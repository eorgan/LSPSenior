# LinhasArquivo

## Assinatura

```lspt
Funcao LinhasArquivo(caminhoNomeArquivo, linhasEmBranco);
```

## Código
N/A

## Descrição

Leitura da quantidade de linhas existentes em um determinado arquivo.

## Parâmetros

- **caminhoNomeArquivo** - Entrada: Caminho onde está localizado o arquivo que deve ser lido.
- **linhasEmBranco** - Entrada: 1 – Ignorar linhas em branco ao final;  2 – Ignorar linhas em branco em todo o arquivo.

## Exemplo de Uso

```lspt
definir Alfa xNome;
definir Alfa CaminhoArquivo;
definir Alfa NomeArquivo;
xNome = CaminhoArquivo + NomeArquivo;
xTeste = LinhasArquivo(xNome, 0);
```
