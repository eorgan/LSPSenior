## Assinatura

## Código

## Descrição

Define se o cursor aplica filtro de abrangência (empresa/filial) automaticamente. Use 0 para desligar e 1 para ligar.

## Parâmetros

- **Cursor** (`Alfa`) - Entrada: Identificador do cursor.
- **Abrangencia** (`Numero`) - Entrada: 0 = sem abrangência; 1 = com abrangência.
