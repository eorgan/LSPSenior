# LimpaGerTabNum

## Assinatura

```lspt
Funcao LimpaGerTabNum();
```

## Código
536

## Descrição

Limpa o conteúdo do Registro GerTabNum.

## Parâmetros

_Sem parâmetros_

## Exemplo de Uso

```lspt
GerTabNum[1] = 1;
LimpaGerTabNum();
@ Todos os elementos do GerTabNum serão limpos @
```
