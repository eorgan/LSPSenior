# DeletarAlfa

## Assinatura

```lspt
Funcao DeletarAlfa (Variavel/Campo, Posicao, Quantidade);
```

## Código
N/A

## Descrição

Remove uma quantidade específica de caracteres de uma posição determinada.

## Parâmetros

- **Variavel/Campo** - Entrada: Quantidade de caracteres que devem ser deletados a partir da posição informada
- **Posicao** - Entrada
- **Quantidade** - Entrada: Quantidade de caracteres que devem ser deletados a partir da posição informada

## Exemplo de Uso

```lspt
Definir Alfa exemplo;
exemplo = "texto de exemplo";
DeletarAlfa (exemplo,1,6);
@ Após o uso da função DeletarAlfa o conteúdo da variável "exemplo" seria "de exemplo". @
```
